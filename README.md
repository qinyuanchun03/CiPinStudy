**新华洞察 (Xinhua Insight)** 是一个专为解构“八股文”设计的深度新闻研判工具。它不仅统计词频，更通过大语言模型（LLM）穿透官方通稿的修饰性辞令，还原背后的政治焦虑、政策动向与社会实质。

v1.1 版本引入了**全中文化 Prompt 系统**，确保在深度研判时逻辑更严密，且完全避免了 AI 输出长篇英文的问题。

---

## 🚀 v1.1 新特性 (New Features)

### 1. 🧠 全中文化逻辑解码
所有 AI 提示词（Prompts）均经过深度中文优化，精准识别政经术语。例如：
*   识别“**稳中求进**”背后的稳增长压力。
*   解析“**统筹安全与发展**”中的监管转向信号。

### 2. 🔍 单篇深度穿透 (Deep Analysis)
不再局限于标题列表分析。点击文章旁的“**深度研判**”，系统将抓取全文并结合选定视角进行 5 维度剖析：
*   **表面意思**: 官方宣传摘要。
*   **深层逻辑**: 穿透式意图解析。
*   **实质影响**: 对普通个体或行业的真实后果。
*   **关键信号**: 提取最能印证逻辑的原句。
*   **语调核查**: 判断通稿是防御性的、动员性的还是警告性的。

### 3. 📦 批量研判与情报归档 (Batch & Dossier)
*   **批量模式**: 支持一次选择多篇文章进行自动化深度分析。
*   **情报档案库**: 分析结果可一键存入“情报档案”，支持永久保存、导出 JSON 或 TXT，方便撰写内参或长期观察。

### 4. 📊 词频风向标优化
内置 `Intl.Segmenter` 分词算法，配合动态更新的“政经热词库”，精准提取真正具有政策指向意义的关键词，剔除无意义的结构性虚词。

## 🤖 五大解码视角 (Persona System)

| 视角 | 解码逻辑 (Decoding Logic) |
|------|--------------------------|
| 🗣️ **大白话 (推荐)** | **翻译官**。将“逆周期调节”翻译成“印钞预警”，将“财政紧平衡”翻译成“存钱防风险”。直接告诉普通家庭怎么做。 |
| 🏃‍♂️ **现实生存 (实用)** | **生存专家**。关注人身安全、物资储备、边境管控。识别“稳定”背后的动荡预警。 |
| 📉 **防御性理财 (空头)** | **宏观分析师**。看财政缺口而非增长目标。关注地方债雷爆、税收倒查与资产贬值风险。 |
| 🕵️ **政治观察员** | **听床师**。分析“谁出席、谁缺席、提法变动”。将平淡的会议通稿还原为激烈的路线抉择。 |
| 🎓 **考公考研 (上岸)** | **申论教练**。提取考点与标准提法。分析政策重心以推断各部门招录趋势（谁在扩权，谁在收缩）。 |

## 🛠 技术实现 (Technical Architecture)

*   **纯客户端架构**: 零后端，API Key 与情报数据仅保存在浏览器 `LocalStorage` 中。
*   **跨域采集**: 利用多路 CORS 代理 (`AllOrigins`, `cross.250221.xyz` 等) 动态抓取新华网移动端源码。
*   **BYOK 模式**: 完美适配 OpenAI (GPT-4o), DeepSeek (V3/R1) 以及本地 Ollama 模型。
*   **分词与清洗**: 结合 N-gram 算法与自定义停用词库，实现秒级的本地新闻文本分析。

## ⚙️ 配置指南

点击右上角 **设置 (⚙️)**：
1. **OpenAI/DeepSeek**: 输入 API Key 和模型 ID (如 `gpt-4o`, `deepseek-chat`)。
2. **中转服务**: 在 Base URL 中填入对应的 API 端点。
3. **本地 Ollama**: 选择 `Local Ollama`，填写本地模型名 (如 `qwen2.5:7b`)。
